# NLU
### Trainning models nlu
`python -m rasa train nlu`
### Run cmd models nlu
`python -m rasa shell nlu`

# Core
### Run server action
`python -m rasa run actions`
### Trainning models
`python -m rasa train`
### Run cmd models
`python -m rasa shell`
### Run server
`python -m rasa run --enable-api --cors "*"`

### Run api-webchat
`python -m rasa run -m models --enable-api --cors "*" --debug`
